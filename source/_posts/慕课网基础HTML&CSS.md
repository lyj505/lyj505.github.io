# 慕课网基础教程总结
## 几个不常见的标签

  1. ```<q></q>```给其中的元素的外面添加引号。```<blockquote></blockquote>```给内容加缩进样式。
  ```<address></address>```语义是地址,给内容加上斜体。

  2. table标签有个属性summary表示摘要的内容,摘要的内容是不会在浏览器中显示出来的。它的作用是增加表格的可读性(语义化),使搜索引擎更好的读懂表格内容,还可以使屏幕阅读器更好的帮助特殊用户读取表格内容。

  3. title属性的作用，鼠标滑过链接文字时会显示这个属性的文本内容。这个属性在实际网页开发中作用很大，主要方便搜索引擎了解链接地址的内容(语义化更友好)。<!--more-->

  4. 所有表单控件（文本框、文本域、按钮、单选框、复选框等）都必须放在```<form></form>```标签之间（否则用户输入的信息可提交不到服务器上哦！

  5. 通过<b>label</b>标签中放入<b>input</b>标签来隐式地连接起来的。显示连接指的就是<b>label</b>标签在前面,<b>input</b>标签在后面,然后通过<b>label</b>标签的<b>for</b>属性和<b>input</b>标签的name属性相等联系起来。

  6. 关于选择符,有通配选择符、类型选择符、类选择符、ID选择符、包含选择符、子对象选择符。
       如   ``` *、p(ul)、.sec、#sec、.sec .sec_1、.sec>.sec_1 ```,其中需要注意的就是包含选择符和子选择符的区别：
  子对象选择符设置的属性只能影响到他子级，“孙级”开始，就没作用了。而包含选择符，就是不管是子级还是孙级，都能影响到。还有一点重要的是:两个类选择器之间存在空格则构成后代选择器，之间没有空格的则构成多类选择器。
  7. HTML标签的权值为1，类选择符的权值为10，ID选择符的权值最高为100,内联样式表的权值最高为1000。
  8. sans-serif无衬线字体，是一类字体，它在操作系统或者浏览器里是可以设置的，你可以把它设置成宋体，也可以设置成微软雅黑，而设置的这种字体肯定是当前系统里存在的字体，所以使用这个字体就一肯能显示出来，所以加上sans-serif就能保证调用。
  9.  em单位的元素具有继承性,继承父辈元素font-size;rem单位的元素具有一致性,继承html根元素的font-size的大小。注意的是，为了兼容不支持 rem 的浏览器，我们需要在 rem 前面写上对应的 px 值，这样不支持的浏览器可以优雅降级。
  10. 绝对定位爆裂拉伸特性可以使图片在容器内居中(不同原理相同效果):

.Absolute_1{position:absolute;top:0;bottom:0;left:0;right:0;margin:auto;}
.Relative_1{position:relative;top:50%;left:50%;margin-left:-50px;margin-top:-50px;width:100px;height:100px;}
